# Discord Community Management Bot

Telegram and Discord bot for online community managers automating member onboarding, rule enforcement, and engagement tracking. Features automatic welcome messages with role assignment, keyword-triggered moderation, daily active member digest, weekly top contributor leaderboard, and event reminder broadcasts. Includes anti-spam detection using message frequency analysis and automatic timeout escalation. Integrates with Patreon for paid tier role syncing. Zero-coding setup via config JSON.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
